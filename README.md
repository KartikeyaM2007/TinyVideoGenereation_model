# AnomVidGen

Low VRAM anomaly image and 2 frame video generation using compact PyTorch diffusion models.

## Supported classes

- Normal
- Fighting
- RoadAccidents

## Notes

This is not a general text-to-video model. Text prompts are mapped to fixed anomaly classes.
